# Promisions
There are three permission types:

| type    | char | digit |
| ------- | ---- | ----- |
| read    | r    | 4     |
| write   | w    | 2     |
| execute | x    | 1     |
|         |      |       |

Permisions are given to:	
![[Promisions1.png]]
1. Owner of a file (creator)
2. Owner group
3. Other users

To add permisions to a file use '+' sign and '-' to remove.
